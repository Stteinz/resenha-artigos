# 📝 Resenhas Críticas — Projeto de Software

Este repositório reúne **resenhas críticas de artigos** da área de Engenharia de Software, desenvolvidas como requisito para a disciplina de **Projeto de Software** do curso de Engenharia de Software da PUC Minas.

## 🎯 Sobre o Projeto

O objetivo deste repositório é centralizar as análises críticas de artigos acadêmicos e técnicos relevantes para a disciplina. Cada resenha sintetiza conceitos fundamentais, contrasta abordagens e explora as implicações práticas dos temas abordados.
